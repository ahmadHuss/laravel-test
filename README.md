# Database
**PostgresSQL**
In `.env` file please put the name of schema according to your own
credentials I have something like this.
```
DB_CONNECTION=pgsql
DB_HOST=127.0.0.1  
DB_PORT=5432
DB_DATABASE=shop_test
DB_USERNAME=postgres
DB_PASSWORD=a
```
